Protocols

The following protocols are available globally.

  • WKInstantiableViewController

    All WKViewController, WKTableViewController, WKCollectionViewController, WKNavigationController conform to this protocol to achieve the static instantation feature.

    All WKViewController, WKTableViewController, WKCollectionViewController conform to this protocol to follow MVP pattern.

    See more

    The base protocol Error of the web requests of the Framework.

    Declaration

    public protocol WKError: Error
  • WKPropertiesInspectable

    Define which properties will be inspectable in the custom WKView. If you add a new property in this protocol, coformed classes will add the property in the UIView extension exactly like the following example:

    @IBInspectable open var theNewVar: CGFloat {
    get {
        // reutrn the value
    }
    set {
        // set the new value
    }
}

    Important

    You must add the annotation @IBInspectable to show the control in the InterfaceBuilder!
    See more

    Swift

    public protocol WKPropertiesInspectable: class
