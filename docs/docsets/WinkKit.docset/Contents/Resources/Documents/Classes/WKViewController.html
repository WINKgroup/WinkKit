WKViewController

open class WKViewController: UIViewController, WKBaseViewController

The base ViewController that will be subclassed in your project instead of subclassing UIViewController. This provides some useful methods like the static instantiation.

Important

Please, if your subclassed WKViewController is from a UIStoryboard and you want to instantiate it by code, USE instantiate(bundle:) or instantiate(fromStoryboard:bundle:) of WKBaseViewController.
