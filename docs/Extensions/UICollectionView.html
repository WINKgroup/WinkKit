UICollectionView

Undocumented

  • register(cell:)

    Registers a WKCollectionViewCell object as cell if the cell view is into a xib. This method internally calls the register(_:forCellWithReuseIdentifier:) of the UICollectionView. Since it uses a UINib, the nibName MUST be the same of the class name!

    Parameter cell: The WKCollectionViewCell to be registered.
